# GTA Companion Brand Guide

## Brand Personality

- Premium
- Modern
- Fast
- Clean
- Reliable
- Exciting
- Player First

## Design Principles

- Simple beats complicated.
- Performance comes before effects.
- Every animation has a purpose.
- Mobile first.
- Build for millions of users.
- Dark mode by default.

## User Experience Goal

When a player opens GTA Companion, the first thought should be:

"Wow... this feels premium."

The second thought should be:

"This is exactly what I needed."

## Motto

Build for players first.
