# GTA Companion Vehicle Database

## Data Policy

Only include vehicles that are:

- Officially shown by Rockstar Games.
- Identified from official trailers.
- Identified from official screenshots.
- Identified from official artwork.

Do not include leaks or unverified vehicles.
