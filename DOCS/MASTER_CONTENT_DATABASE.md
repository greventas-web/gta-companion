# Development Workflow

Every new feature follows this order.

1. Research

Verify using official Rockstar Games sources.

2. Master Database

Add or update the entry in this document.

3. Assets

Collect official screenshots, artwork and logos.

4. Data

Update the TypeScript data files.

5. Components

Build reusable UI components.

6. Pages

Create or update the page.

7. Internal Links

Connect related pages together.

8. SEO

Add metadata and structured data.

9. QA

Test desktop, tablet and mobile.

10. Publish

Deploy after verification.
